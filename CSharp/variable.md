1. 정수형 : 대충 숫자, 그런데 소수는 뺴고, 앞에 U였나? 그거 붙이면 + 만 된다던데... 애매하게 쓰인다한다, - 들어가면 이상한, 랭킹같은 부분에 들어간다고 
1. 소수형 : 소수가 들어간다, 많읻 ㅡㄹ어간다, 대신 가끔 뒷자리가 다른데도 같다고 오류내는 경우가 있어 정확하게 계상해야하는 은행같은데서 쓰려고 만든 특수 소수 변수가 있다고 어디서 봤는데 까먹었다
1. 이름 까먹었지만 일단 bool : 얘는 뭐 없고 맞고 틀림만 본다, 그거랑 '아무것도 안들음' 만 해서 3개만 있다
1. 문자명 char : 문자 표시용인데 얘가 숫자 넣으면 그자리 문자 뽑아오는거라 숫자기도 하다
1. String 문자열 : 길이 제한이 없는 대신 1회용이다, 새로 넣으면 넣는 대신 새거 이름 같게 만들어거 것다가 너ㅗㅎ는다
1. [] 이거 엇다 넣을가 생각하다가 일단 여기 넣기로함, 변수 종류 뒤에 붙이면 원하는 방이 들어있는 상자 만들어준다, int[5] 면 int용 방 5개 들은 층 주는거임, 대신 방 번호 내가 찾아서 거기에 넣어야함
