
- 이건 진짜 모르겠다
- 그냥 둘다 컴푸터 부품 아닌가, 일단 CPU가 더 익수ㅡㄱ하군
- 계산기기였나
- 전에 쌤이 나눠준거 봤지만 잘 모르겠다, 다 읽고 몇번더 봐야지

1. CPU
>- 뭔가 CPU파는 사이트에서 설명문 찾았다, 저런 놈들이 설명 열심히 적어두지, 그래야 잘 팔리니까
>- 대충, CPU는 컴퓨터의 계싼이기고 저장소들 관리하고 꺼내서 쓰고 뭐 대충 중요한거 다 한덴다
>- 공통적으로 나오는 비유가 컴퓨터의 두뇌라는거 보니까 진짜 중요한가보다


2. GPU 
> - 대충 그래픽 처리용으로 반복 같은작업 계속 반복용 무언가
> - 공장형이긴한데 뭔가 약~간 다른것들 처리 잘하는애
> - AI 뇌세포, 이런저런거 잡다하게 잘하는듯

3. CPU 와 GPU의 차이
> - 알기쉽게 말해 다빈치와 공장기계 차이
> - 다빈치 = 천재여서 온갇분야 다 잘하는데 여러개 동시에 잘 못함
> - 공장 기계 = 시키는 일 하나 잘하는데 그 분야 벗어나면 다른 기계 서야함
> - 좀 더 다르게 말하자면... 지휘자와 재봉기계?
> - 그니까 결국 일많고 머리아픈 만능이랑 자기영역만 잘하는 반복재생이네
